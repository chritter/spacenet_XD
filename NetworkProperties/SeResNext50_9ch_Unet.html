Neural Network Properties of UNet with VGG16¶

Goals¶

  • The goal is to understand the network properties
  • Estimate flops and compute time before execution
  • Estimate memory usage before computation
  • Estimate time of forward pass through network

Comments¶

  • All this information will help to commpare different models
  • UNet only uses 3 channels! and its output is just 1 channel
